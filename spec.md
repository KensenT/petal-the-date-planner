## Problem Statement
Sometimes when my girlfriend and I go on a date, we want to visit multiple places on the same day. To make sure that we don't waste too many times during the travel, we planned our route using google maps. We put the locations we want to visit using the add stops feature, then rearrange them until we are satisfied with the total estimated time. This process could take 10-15 minutes.

## Objective
Create a solution to automate / simplify this process.

## Constraints
* Solution must be in html form.
* Solution must be able to be deployed for public uses.
